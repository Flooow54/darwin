Sur les épaules de Darwin, Jean-Claude Ameisen

Cette page (sur laquelle je travaille à mes heures perdues) agrège tous les liens vers les épisodes de l'émission : page de l'épisode, page pour ré-écouter l'épisode, lien direct vers les fichier mp3.

  • Les scripts utilisés pour la génération automatique de cette page sont accessibles sur GitHub : https://github.com/cgrimal/darwin
  • Si vous voulez en savoir plus sur la création automatique de cette page, direction mon blog : http://clementgrimal.fr/blog/tag/darwin/
  • Vous pouvez laisser vos commentaires plus bas sur cette page.
  • Vous pouvez aussi rejoindre le groupe presque officiel des fans de l'émission sur Facebook.
  • (non maintenu) Vous pouvez facilement télécharger plusieurs émissions d'un coup via ce dossier MEGA
    (merci à Paul pour l'idée initiale, et retrouvez son découpage des épisodes sur son dossier MEGA)
